# Smart Hire 🛠️

Smart Hire is an on-demand local services platform connecting customers with professional service providers (plumbers, electricians, cleaners, etc.).

---

## 📁 Project Structure

* **Frontend**: [`/frontend`](./frontend) — Web interface hosted on Vercel (`https://frontend-weld-one-61.vercel.app`)
* **Backend**: [`/backend`](./backend) — FastAPI REST API hosted on Vercel (`https://backend-ashy-six-26.vercel.app`)
